# Blog Backend

A personal blog system backend built with Go, Gin and GORM.

## Features

- User authentication (JWT)
- CRUD operations for blog posts
- Comment system
- RESTful API design

## Prerequisites

- Go 1.20+
- SQLite (or other supported database)
- Docker (optional)
